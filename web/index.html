快速开始

3个简单步骤，即刻上手

通过直观的命令行界面，几分钟内即可开始分析你的代码库

1

安装 CodeSnapAI

pip install codesnapai
# 或从源码安装
git clone https://github.com/turtacn/CodeSnapAI.git
cd CodeSnapAI
poetry install
已复制！

要求 Python 3.10+ 版本，Git（用于仓库分析功能）

2

分析你的代码库

# 初始化配置
codesnapai config init --interactive

# 分析项目
codesnapai analyze ./your-project
已复制！

分析器自动检测项目语言，并生成完整的语义模型

3

生成快照并调试

# 创建语义快照
codesnapai snapshot create -o snapshot.yaml

# 自然语言调试
codesnapai debug snapshot.yaml "为什么TestLogin测试失败？"
已复制！

获取AI驱动的问题洞察和可直接落地的代码修复方案
